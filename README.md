YourCabs Cancellation Prediction 

This repository contains my machine learning project focused on **predicting cab booking cancellations** using data analysis and classification models.

Overview:-
The project analyzes booking data from YourCabs and builds models to predict whether a ride will be cancelled or not. It includes data preprocessing, feature engineering, and model training.

Key Steps Covered:-
* Data cleaning and preprocessing
* Handling missing values and categorical data
* Exploratory Data Analysis (EDA)
* Feature selection and transformation
* Model building (classification models)
* Model evaluation and performance analysis

Models Used:-
* Logistic Regression
* Decision Tree
* Random Forest
* XG Boost

How to Run:-
1. Open the notebook in Jupyter Notebook / JupyterLab / VS Code
2. Run all cells step by step
3. Observe data processing, model training, and evaluation

Objective:-
The goal of this project is to:
* Predict ride cancellations in advance
* Improve understanding of customer behavior
* Apply machine learning to real-world problems

Future Improvements:-
* Hyperparameter tuning for better accuracy
* Try advanced models (XGBoost, Gradient Boosting)
* Deploy the model as a web application
